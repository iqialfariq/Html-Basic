## Tag Dasar HTML

- **Tag Heading h1 – h6**  
  Contoh: `<h1> Ini Tag Heading </h1>`

- **Tag Paragraph**  
  Contoh: `<p> Ini Tag Paragraf </p>`

- **Tag Bold**  
  Contoh: `<b> Ini Teks Tebal </b>`

- **Tag Italic**  
  Contoh: `<i> Ini Teks Miring </i>`

- **Tag Superscript**  
  Contoh: `<sup> Ini Superscript </sup>`

- **Tag Subscript**  
  Contoh: `<sub> Ini Subscript </sub>`

- **Tag Strong**  
  Contoh: `<strong> Ini Strong </strong>`

- **Tag Emphasized**  
  Contoh: `<em> Ini Emphasized </em>`

- **Tag Anchor / Link**  
  Contoh: `<a href="link"> Ini Tag Link </a>`

- **Tag Image**  
  Contoh: `<img src="Folder" alt="Deskripsi gambar" />`

- **Tag Div**  
  Contoh: `<div> Konten Besar </div>`

- **Tag Span**  
  Contoh: `<span> Konten Penting </span>`

- **Tag Horizontal Rule**  
  Contoh: `<hr>`

- **Tag Break Line**  
  Contoh: `<br>`

- **Tag Button**  
  Contoh: `<button> Click Me </button>`

- **Tag Input**  
  Contoh: `<input type="text">`  
  (type bisa: radio, checkbox, submit, text, button)

- **Tag Form**  
  Contoh: `<form> Konten Form </form>`

- **Tag Label**  
  Contoh: `<label> Penanda </label>`

- **Tag Video**  
  ```html
  <video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
  </video>
- **Tag Audio**  
  ```html
  <audio controls>
    <source src="horse.ogg" type="audio/ogg">
  </audio>

- **Tag List**  
  Unorder vs Order List  

  ```html
  <ul>
    <li> Unorder List </li>
    <li> Unorder List </li>
  </ul>

  <ol>
    <li> Order List </li>
    <li> Order List </li>
  </ol>

- **Tag Table**  
  Colspan: Menggabungkan beberapa kolom  
  Rowspan: Menggabungkan beberapa baris  

  ```html
  <table>
    <tr>
      <th> Table Heading </th>
      <th> Table Heading </th>
      <th> Table Heading </th>
    </tr>
    <tr>
      <td> Table Data </td>
      <td> Table Data </td>
      <td> Table Data </td>
    </tr>
  </table>
