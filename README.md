## Tag Dasar HTML

- **Tag Heading h1 – h6**  
  ```html
  <h1> Ini Tag Heading </h1>

- **Tag Paragraph**  
  ```html
  <p> Ini Tag Paragraf </p>

- **Tag Bold**  
  ```html
  <b> Ini Teks Tebal </b>

- **Tag Italic**  
  ```html
  <i> Ini Teks Miring </i>

- **Tag Superscript**  
  ```html
  <sup> Ini Superscript </sup>

- **Tag Subscript**  
  ```html
  <sub> Ini Subscript </sub>

- **Tag Strong**  
  ```html
  <strong> Ini Strong </strong>

- **Tag Emphasized**  
  ```html
  <em> Ini Emphasized </em>

- **Tag Anchor / Link**  
  ```html
  <a href="link"> Ini Tag Link </a>

- **Tag Image**  
  ```html
  <img src="Folder" alt="Deskripsi gambar" />

- **Tag Div**  
  ```html
  <div> Konten Besar </div>

- **Tag Span**  
  ```html
  <span> Konten Penting </span>

- **Tag Horizontal Rule**  
  ```html
  <hr>

- **Tag Break Line**  
  ```html
  <br>

- **Tag Button**  
  ```html
  <button> Click Me </button>

- **Tag Input**  
  ```html
  <input type="text">

- **Tag Form**  
  ```html
  <form action="/submit" method="post">
    <!-- Isi form di sini -->
  </form>

- **Tag Label**  
  ```html
  <label for="inputId"> Nama Field </label>

- **Tag Video**  
  ```html
  <video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
  </video>
- **Tag Audio**  
  ```html
  <audio controls>
    <source src="horse.ogg" type="audio/ogg">
  </audio>

- **Tag List**  
  Unorder vs Order List  

  ```html
  <ul>
    <li> Unorder List </li>
    <li> Unorder List </li>
  </ul>

  <ol>
    <li> Order List </li>
    <li> Order List </li>
  </ol>

- **Tag Table**  
  Colspan: Menggabungkan beberapa kolom  
  Rowspan: Menggabungkan beberapa baris  

  ```html
  <table>
    <tr>
      <th> Table Heading </th>
      <th> Table Heading </th>
      <th> Table Heading </th>
    </tr>
    <tr>
      <td> Table Data </td>
      <td> Table Data </td>
      <td> Table Data </td>
    </tr>
  </table>
