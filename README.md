# 🧠 Penjelasan HTML
<p align="justify">
HTML (HyperText Markup Language): adalah bahasa markah standar untuk membuat halaman web. Html sendiri ditemukan oleh Tim Berners-Lee.
</p>

---

## 📌 Macam-macam Tag Html

### 🔠 1. Tag Heading h1 – h6
  ```html
  <h1> Ini Tag Heading </h1>
  ```
---  

### 📜 2. Tag Paragraph
  ```html
  <p> Ini Tag Paragraf </p>
  ```
---  

### 💪 3. Tag Bold
  ```html
  <b> Ini Teks Tebal </b>
  ```
---  

### 🖊️ 4. Tag Italic
  ```html
  <i> Ini Teks Miring </i>
  ```
---  

### 🔼 5. Tag Superscript
  ```html
  <sup> Ini Superscript </sup>
  ```
---  

### 🔽 6. Tag Subscript
  ```html
  <sub> Ini Subscript </sub>
  ```
---  

### 🏋️ 7. Tag Strong
  ```html
  <strong> Ini Strong </strong>
  ```
---  

### 🖊️ 8. Tag Emphasized
  ```html
  <em> Ini Emphasized </em>
  ```
---  

### 🔗 9. Tag Anchor / Link
  ```html
  <a href="link"> Ini Tag Link </a>
  ```
---  

### 🖼️ 10. Tag Image
  ```html
  <img src="Folder" alt="Deskripsi gambar" />
  ```
---  

### 📦 11. Tag Div
  ```html
  <div> Konten Besar </div>
  ```
---  

### 🔖 12. Tag Span
  ```html
  <span> Konten Penting </span>
  ```
---  

### ➖ 13. Tag Horizontal Rule
  ```html
  <hr>
  ```
---  

### ↩️ 14. Tag Break Line
  ```html
  <br>
  ```
---  

### 🔘 15. Tag Button
  ```html
  <button> Click Me </button>
  ```
---  

### 🔤 16. Tag Input
  ```html
  <input type="text">
  ```
---  

### 📝 17. Tag Form
  ```html
  <form action="/submit" method="post">
    <!-- Isi form di sini -->
  </form>
  ```
---

### 🏷️ 18. Tag Label
  ```html
  <label for="inputId"> Nama Field </label>
  ```
---  

### 🎬 19. Tag Video
  ```html
  <video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
  </video>
  ```
---

### 🎵 20. Tag Audio 
  ```html
  <audio controls>
    <source src="horse.ogg" type="audio/ogg">
  </audio>
  ```
---  

### 📋 21. Tag List
  Unorder vs Order List  

  ```html
  <ul>
    <li> Unorder List </li>
    <li> Unorder List </li>
  </ul>

  <ol>
    <li> Order List </li>
    <li> Order List </li>
  </ol>
  ```
---

### 📊 22. Tag Table 
  Colspan: Menggabungkan beberapa kolom  
  Rowspan: Menggabungkan beberapa baris  

  ```html
  <table>
    <tr>
      <th> Table Heading </th>
      <th> Table Heading </th>
      <th> Table Heading </th>
    </tr>
    <tr>
      <td> Table Data </td>
      <td> Table Data </td>
      <td> Table Data </td>
    </tr>
  </table>
  ```
